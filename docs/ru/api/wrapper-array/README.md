# WrapperArray

`WrapperArray` — это объект, содержащий массив [`Wrappers`](../wrapper/README.md), и методы для тестирования `Wrappers`.

- **Свойства:**

`length` `number`: the number of `Wrappers` contained in the `WrapperArray`
