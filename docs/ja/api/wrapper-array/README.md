# WrapperArray

`WrapperArray` は、[Wrapper](../wrapper/) の配列と `Wrapper` をテストするメソッドを含むオブジェクトです。

## プロパティ

### `wrappers` 

`array`: `WrapperArray` に含まれる `Wrappers`  

### `length` 

`number`: `WrapperArray` に含まれる `Wrappers` の数
