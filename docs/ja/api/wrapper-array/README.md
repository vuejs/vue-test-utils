# WrapperArray

`WrapperArray` は、[Wrapper](../wrapper/) の配列と `Wrapper` をテストするメソッドを含むオブジェクトです。

## プロパティ

### `wrappers`

`array` (読み込み専用): `WrapperArray` に含まれる `Wrappers`

### `length`

`number` (読み込み専用): `WrapperArray` に含まれる `Wrappers` の数
