コンポーネント

vue-test-utils にはコンポーネントをスタブするためのユーティリティコンポーネントがあります。

TransitionStub と TransitionGroupStub はデフォルトで transition コンポーネントと transition-group コンポーネントをスタブすることに使用されます。 これらのスタブは config で変更することができます。
